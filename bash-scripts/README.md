# Bash Scripts

Collection of automation and system administration scripts.

## organize_files.sh

**Purpose:** Automatically organizes files in a directory by type

**Features:**
- Creates folders for documents, images, scripts, archives, and videos
- Moves files based on extension
- Handles errors gracefully

**Usage:**
```bash
./organize_files.sh
```

## sysinfo.sh

**Purpose:** Displays comprehensive system information

**Features:**
- Shows current user and directory
- Displays date and time
- CPU model information
- Running process count
- Disk usage statistics
- Memory usage statistics

**Usage:**
```bash
./sysinfo.sh
```
